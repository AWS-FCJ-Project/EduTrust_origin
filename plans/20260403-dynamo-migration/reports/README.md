# Reports

Thư mục này dùng để lưu report thực thi của từng phase hoặc từng agent.

## Format khuyên dùng
- `YYMMDD-from-[agent]-to-[agent]-[task]-report.md`

## Ví dụ
- `260403-from-main-to-main-phase-01-persistence-refactor-report.md`
- `260403-from-tester-to-main-phase-04-parity-check-report.md`

## Nội dung tối thiểu của mỗi report
- Phase hoặc task đã thực hiện
- File đã sửa
- Kết quả kiểm tra
- Vấn đề còn mở
- Quyết định kỹ thuật đã chốt trong quá trình làm
